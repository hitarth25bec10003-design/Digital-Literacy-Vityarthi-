# Digital-Literacy-Vityarthi-

Project Overview –

This repository contains my Digital Literacy Portfolio created as part of the CSE0001 – Digital Literacy course at VIT Bhopal University.
As a Student Digital Ambassador, the objective of this project is to help peers understand and navigate the digital world responsibly. The project includes five tasks covering essential digital literacy skills such as professional online presence, collaboration tools, email etiquette, and cybercrime awareness.
